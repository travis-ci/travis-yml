Sonarcloud

Sonarcloud settings

See also

  • Using SonarCloud with Travis CI

Types

  • Map
  • Boolean

Keys

If given a map, the following keys are supported:

  • enabled — (type: boolean)

  • organization — (type: string)

  • token — (type: secure)

  • github_token — (deprecated, type: secure)

  • branches — (deprecated, type: sequence of string, or string)

Examples

sonarcloud:
  enabled: true
  organization: string
  token:
    secure: encrypted string

sonarcloud: true
