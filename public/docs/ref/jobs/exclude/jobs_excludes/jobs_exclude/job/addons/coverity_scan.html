Coverity Scan

CoverityScan settings

See also

  • The Coverity Scan Addon for Travis CI

Types

  • Map
  • Boolean

Keys

If given a map, the following keys are supported:

  • project — (type: map, or string)

  • build_script_url — (type: string)

  • branch_pattern — (type: string)

  • notification_email — (type: secure)

  • build_command — (type: string)

  • build_command_prepend — (type: string)

Examples

coverity_scan:
  project:
    name: string
    version: string
    description: string
  build_script_url: string
  branch_pattern: string

coverity_scan: true
