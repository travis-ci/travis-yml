# Deploy S3



## Type

Any of:

* Map (Hash)
* Enum (known String)

## Flags

None.


## Examples

```yaml
deploy:
  provider: s3
  on: {}
  allow_failure: true
  skip_cleanup: true
  edge: true
  access_key_id: 
  secret_access_key: 
  bucket: string
  region: string
  upload_dir: string
  storage_class: string
  local_dir: string
  detect_encoding: true
  cache_control: string
  expires: string
  acl: string
  dot_match: true
  index_document_suffix: string
  default_text_charset: string
  server_side_encryption: true
```

```yaml
deploy: s3

```
